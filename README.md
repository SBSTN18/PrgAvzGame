# EL JUEGO DEL BALERO


## Taller 1 - PAVA

Proyecto desarrollado para la asignatura **Programación Avanzada** de Ingeniería de Sistemas.

El taller consiste en diseñar e implementar un aplicativo en **Java**, bajo arquitectura **MVC**, que simule el juego tradicional del **Balero**, cumpliendo con principios **SOLID**, buenas prácticas de programación y manejo de persistencia tanto volátil como no volátil.

El sistema debe trabajar con entorno gráfico, cargar información desde archivos externos, manejar almacenamiento en archivo serializado y utilizar obligatoriamente un **RandomAccessFile** para persistir los resultados de cada ejecución.


<br>

## Descripción del Juego

El Balero es un juego tradicional que consiste en encajar una pieza de madera (mazo o coca) en un eje mediante movimientos de muñeca.

En el sistema:

- Se compite por **equipos de 3 jugadores**
- Cada equipo tiene un tiempo determinado para jugar
- Cada jugador usa un tercio del tiempo asignado
- Las embocadas se generan de forma aleatoria
- Cada tipo de embocada otorga un puntaje distinto
- Se contabilizan tanto los puntos como los intentos
- Gana el equipo con mayor puntaje
- En caso de empate, se define por menor cantidad de intentos realizados
